# Makro_kerdesbank

Randomizált makroökonómia kérdésbank 250 feleletválasztós kérdéssel, azonnali visszajelzéssel és részletes indoklással.

## Tartalom

- A kérdések minden új tesztnél véletlen sorrendben jelennek meg.
- A kérdésbank eredeti sorszáma gyakorlás közben nem látszik.
- Válaszjelölés után az oldal azonnal kiírja a helyes választ, az indoklást és a kizárásos logikát.
- A teszt végén összesítést ad, és felsorolja a hibásan megválaszolt kérdések eredeti kérdésbank-sorszámait.
- A teszt korábban is lezárható egy megerősítő ablak után; ekkor a kihagyott kérdések is bekerülnek a sorszámlistába.
